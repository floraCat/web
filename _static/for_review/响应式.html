媒体查询： ---------------------------------------------- @media : 指定样式表规则用于指定的设备类型。 指定媒体: @media screen : 指定样式用于显示器 @media all : 指定样式用于所有媒体类型 @media print : 指定样式用于打印机 ***特性*** ********** 屏幕可见高度 - device-height ： ---------------------------------------------- @media screen and (device-height:800px){ … } device-height : 定义输出设备的屏幕可见高度。 本特性接受min和max前缀，因此可以派生出min-device-height和max-device-height两个媒体特性。 width,height,device-width(前可加'min-'或'max-')类同上； 可见区域高度是否大于或等于宽度 - orientation ： ---------------------------------------------- @media screen and (orientation:portrait){ … } orientation : 定义输出设备中的页面可见区域高度是否大于或等于宽度。 portrait：指定输出设备中的页面可见区域高度大于或等于宽度 landscape：除portrait值情况外，都是landscape 本特性不接受min和max前缀。 宽高比率 - device-aspect-ratio ： ---------------------------------------------- @media screen and (device-aspect-ratio:4/3){ … } device-aspect-ratio : 定义输出设备的屏幕可见宽度与高度的比率。 如常讲的显示器屏幕比率：4/3, 16/9, 16/10 本特性接受min和max前缀，因此可以派生出min-device-aspect-ratio和max-device-aspect-ratio两个媒体特性。 aspect-ratio类同上； 色彩数 - color ： ---------------------------------------------- @media screen and (color){ … } color : 定义输出设备每一组彩色原件的个数。如果不是彩色设备，则值等于0 与色彩color不同，媒体特性color的取值只能是<integer>，用来表示色彩数。 本特性接受min和max前缀，因此可以派生出min-color和max-color两个媒体特性。 彩色查询表中的条目数 - color-index : ---------------------------------------------- @media screen and (color-index){ … } color-index : 定义在输出设备的彩色查询表中的条目数。如果没有使用彩色查询表，则值等于0 本特性接受min和max前缀，因此可以派生出min-color-index和max-color-index两个媒体特性。 单色原件个数 - monochrome ： ---------------------------------------------- @media screen and (monochrome){ … } monochrome : 定义在一个单色框架缓冲区中每像素包含的单色原件个数。如果不是单色设备，则值等于0 本特性接受min和max前缀，因此可以派生出min-monochrome和max-monochrome两个媒体特性。 分辨率 - resolution ： ---------------------------------------------- @media screen and (resolution){ … } resolution : 定义设备的分辨率。如：96dpi, 300dpi, 118dpcm 本特性接受min和max前缀，因此可以派生出min-resolution和max-resolution两个媒体特性。 扫描工序 - scan ： ---------------------------------------------- @media tv and (scan:progressive){ … } scan : 定义电视类设备的扫描工序。 本特性不接受min和max前缀。 栅格 - grid ： ---------------------------------------------- @media all and (grid){ … } grid : 用来查询输出设备是否使用栅格或点阵。 本特性不接受min和max前缀。 —————————————————————————————————————————————————————————————————————————————————————————————————————————— 注意事项： ---------------------------------------------- ---手机端最小宽度以320px为准，最宽以640px为准； ---ipad分辨率：768px * 1024px； ---水平方向一般用百分比%，垂直方向可参照em单位(默认字体的倍数)； ---弹性图片: img{ max-width:100%}; ---em(相对于父节点的font-size)，rem(相对于html的font-size); ---em一般用作排版元素适配设备的，字体一般不要作为排版元素 //rem=screen.width/20 ?? ---ie8及以下版本不支持Media Query,兼容方法： <!--[if lt IE 9]> <script src="http://css3-mediaqueries-js.googlecode.com/svn/trunk/css3-mediaqueries.js"></script> <![endif]--> ---常用取值： ---240 ---320 ---480 ---640 ---720 ---960 ---1200 ---1280 ---1440 其他： ---------------------------------- 当页面通过屏幕呈现，非打印一类，并且屏幕不超过480px，则加载样式表： <link rel="stylesheet" type="text/css" media="screen and (max-device-width:480px)" href="aaa.css" /> 参考： ---------------------------------- 京东默认body{font-size:12px},.main{font-size:14px;line-height:1.25em}; @media(min-width:640px){.main{font-size:28px;}} @media(min-width:480px){.main{font-size:21px;}} 实例： ---------------------------------- @media screen and(max-width:1024px){...} @media screen and(max-width:980px){...} @media screen and(max-width:720px){...} @media screen and(max-width:640px){...} @media screen and(max-width:320px){...} 未确定： ---------------------------------- @media (min-width: @grid-float-breakpoint){...} //?? @media screen and (-webkit-min-device-pixel-ratio: 0){...} //?? @media only screen and (min-device-width:320) and (max-device-width:480){...} @media all and (-webkit-transform-3d){...}/* Android4.0下不识别该-webkit-transform-3d，使用它可做Android4.0下版本兼容 */ @media all and (transform-3d), (-webkit-transform-3d){} //??
