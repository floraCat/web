clientTop = border-top宽

scrollTop = 滚动了的高度

offsetTop = top + margin-top (相对上一父级)

------------------------------------------------------------------------

clientLeft = border-left宽

scrollLeft = 滚动了的宽度

offsetLeft = left + margin-left (相对上一父级)

------------------------------------------------------------------------

clientHeight = 高度 + padding上下 (不包含border,且减掉滚动轴宽)

scrollHeight = 总滚动高度 (包含padding,包括被隐藏的)

offsetHeight = 高度 + padding上下 + border上下
